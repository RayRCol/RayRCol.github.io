## Scope

Portfolio v2 will expand each project entry with an interactive demo — users will be able to input their own data and receive predictions from deployed models. A live Streamlit dashboard will pull real data from public APIs to demonstrate the Retail Feedback Intelligence pipeline.

## Tech Plan

Deployed model APIs will be hosted on Hugging Face Spaces and Railway. The portfolio front end will embed each demo via iframe with a loading fallback. A public GitHub repository will accompany each project page with clean, documented notebooks.

## Goal

A portfolio that demonstrates not just that models were built, but that they work in production — the highest-signal proof of data science competency for hiring teams.
